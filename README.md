# RayTracerApp
A simple raytracer built with C++

## Features
- Renders spheres
- Support for:
  - Shadows
  - Reflections
- Image output in BMP
- Multithreading

## Built With

- C++17
- CMake
- SFML

## Installation
1. git clone https://github.com/matthew-sanders-code/RayTracerApp.git
2. cd RayTracerApp
3. cmake --build build --config release
4. cd build/Release
5. ./RayTracerApp
